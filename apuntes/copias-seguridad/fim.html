FIM - File Integrity Monitoring

El monitoreo de integridad de archivos (FIM) es una técnica de seguridad que detecta cambios no autorizados en archivos críticos del sistema y aplicaciones.

Funciones Principales

  • Monitoreo de cambios: Detecta modificaciones, eliminaciones o creaciones de archivos
  • Líneas base: Establece estados conocidos de archivos para comparación
  • Alertas en tiempo real: Notifica sobre cambios detectados
  • Análisis de integridad: Verifica la autenticidad de archivos críticos
  • Exclusiones configurables: Permite definir excepciones legítimas

Archivos Monitoreados Típicamente

  • Archivos de configuración del sistema
  • Registros del sistema operativo
  • Archivos ejecutables críticos
  • Bases de datos de configuración
  • Archivos de aplicaciones sensibles
Importancia en cumplimiento: FIM es esencial para estándares como PCI DSS, CIS benchmarks, y otras normativas que requieren monitoreo de integridad de archivos críticos.
