CVE - Common Vulnerabilities and Exposures

CVE es un sistema de identificación de vulnerabilidades de seguridad en software y hardware. Cada vulnerabilidad recibe un identificador único para facilitar el seguimiento y gestión.

Características Principales

  • Identificadores únicos: Cada vulnerabilidad tiene un ID único (ej: CVE-2023-12345)
  • Base de datos centralizada: Repositorio público de vulnerabilidades conocidas
  • Formato estandarizado: Estructura consistente para describir vulnerabilidades
  • Referencias múltiples: Enlaces a información adicional y parches
  • Actualización continua: Nuevos CVEs se agregan regularmente

Componentes de un CVE

  • CVE ID: Identificador único en formato CVE-AAAA-NNNN
  • Descripción: Explicación técnica de la vulnerabilidad
  • Referencias: Enlaces a información adicional
  • Estado: Estado actual (reservado, publicado, rechazado)
  • Fecha de asignación: Cuándo se asignó el identificador
Importancia en Wazuh: Wazuh utiliza bases de datos CVE para escanear vulnerabilidades en sistemas endpoint y proporcionar información detallada sobre riesgos de seguridad.
