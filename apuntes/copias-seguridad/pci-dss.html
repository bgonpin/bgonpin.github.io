PCI DSS - Payment Card Industry Data Security Standard

PCI DSS es un estándar de seguridad de datos establecido por la industria de tarjetas de pago que establece requisitos para proteger la información de titulares de tarjetas.

Objetivos Principales

  • Proteger datos de tarjetas: Mantener la confidencialidad de los datos de pago
  • Prevenir fraudes: Reducir la incidencia de transacciones fraudulentas
  • Establecer estándares: Crear un marco común para la seguridad de pagos
  • Fomentar la confianza: Aumentar la confianza de los consumidores

Los 12 Requisitos Principales

  • Firewall: Instalar y mantener configuraciones de firewall
  • Contraseñas: No usar valores por defecto de proveedores
  • Datos sensibles: Proteger datos de titulares de tarjetas almacenados
  • Datos en tránsito: Cifrar transmisión de datos de tarjetas
  • Antivirus: Usar y mantener software antivirus
  • Software seguro: Desarrollar y mantener sistemas seguros
  • Acceso restringido: Restringir acceso a datos de tarjetas por necesidad
  • Identificación: Identificar y autenticar el acceso a componentes del sistema
  • Acceso físico: Restringir el acceso físico a datos de tarjetas
  • Monitoreo: Hacer un seguimiento y monitoreo del acceso
  • Pruebas de seguridad: Probar regularmente sistemas y procesos
  • Política de seguridad: Mantener una política de seguridad de la información
