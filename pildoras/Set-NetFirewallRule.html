Editor Interactivo de `Set-NetFirewallRule`

Simula la modificación de reglas de firewall existentes de manera visual y práctica.

Selector de Regla

Selecciona una de las siguientes reglas de ejemplo para comenzar a editarla. Esto simula el uso de `Get-NetFirewallRule` para identificar una regla.

Opciones de Edición

Modifica los parámetros de la regla seleccionada. Los cambios se reflejarán en el comando de abajo.

Comando Generado

Este es el comando `Set-NetFirewallRule` que aplicará tus cambios. Primero identifica la regla por su nombre, y luego especifica los parámetros a modificar.

Análisis del Estado

Este gráfico muestra la distribución de las reglas habilitadas y deshabilitadas, una de las operaciones más comunes con este cmdlet.
