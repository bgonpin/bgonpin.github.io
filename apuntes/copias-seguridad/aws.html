AWS - Amazon Web Services

Amazon Web Services (AWS) es la plataforma de cloud computing más utilizada del mundo, que ofrece una amplia gama de servicios de infraestructura, computación, almacenamiento y seguridad.

Servicios Principales de Seguridad

  • IAM (Identity and Access Management): Gestión de identidades y accesos
  • CloudTrail: Registro y monitoreo de actividad de API
  • GuardDuty: Detección inteligente de amenazas
  • Security Hub: Panel centralizado de seguridad
  • WAF (Web Application Firewall): Protección de aplicaciones web
  • CloudWatch: Monitoreo y alertas

Características Clave

  • Escalabilidad global: Infraestructura distribuida en múltiples regiones
  • Servicios gestionados: AWS maneja la infraestructura subyacente
  • Modelo de pago por uso: Solo paga por lo que consume
  • Integración nativa: Servicios diseñados para trabajar juntos
  • Seguridad por defecto: Principio de seguridad en todas las capas
Integración con Wazuh: Wazuh puede integrarse con AWS CloudTrail, GuardDuty y otros servicios para proporcionar monitoreo completo de entornos cloud.
