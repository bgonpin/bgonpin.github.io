¿Qué es iLDAP?

iLDAP (Lightweight Directory Access Protocol) es un protocolo de aplicación utilizado para acceder y mantener servicios de información de directorio distribuida sobre una red IP. Es una versión simplificada del protocolo DAP (Directory Access Protocol) que forma parte del protocolo X.500.

Características principales

  • Protocolo cliente-servidor: Arquitectura basada en peticiones y respuestas
  • Operaciones principales: Búsqueda, adición, modificación y eliminación de entradas
  • Autenticación: Soporte para diversos mecanismos de autenticación
  • Seguridad: Puede operar sobre SSL/TLS (LDAPS)

Usos comunes

  • Autenticación centralizada de usuarios
  • Gestión de directorios de empresas
  • Servicios de correo electrónico
  • Integración con sistemas de copias de seguridad

Implementaciones populares

  • OpenLDAP: Implementación de código abierto más utilizada
  • Microsoft Active Directory: Basado en LDAP para Windows
  • IBM Tivoli Directory Server
  • Oracle Directory Server

Comandos básicos LDAP

# Búsqueda en directorio LDAP
ldapsearch -x -b "dc=example,dc=com" -H ldap://localhost

# Adición de entrada
ldapadd -x -D "cn=admin,dc=example,dc=com" -W -f entrada.ldif

# Modificación de entrada
ldapmodify -x -D "cn=admin,dc=example,dc=com" -W -f cambios.ldif
