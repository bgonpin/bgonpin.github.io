Windows Sandbox

Un entorno seguro y aislado para probar aplicaciones en Windows

¿Qué es Windows Sandbox?

Windows Sandbox es un entorno de escritorio virtualizado, ligero y aislado integrado en Windows que permite ejecutar aplicaciones de forma segura sin afectar al sistema principal.

Consejo: Todas las aplicaciones y archivos utilizados en Windows Sandbox se eliminan permanentemente cuando se cierra la aplicación, manteniendo tu sistema seguro.

Características principales

  • Incluido en Windows: Integrado en Windows 10/11 Pro, Enterprise y Education
  • Inmutable: Cada sesión es completamente nueva y limpia
  • Seguro: Utiliza virtualización basada en hardware para aislamiento
  • Eficiente: Utiliza la tecnología de integración inteligente del kernel
  • Rápido: Se inicia en cuestión de segundos

Requisitos del sistema

Arquitectura

AMD64 o ARM64

Virtualización

Activada en BIOS/UEFI

Memoria RAM

4 GB mínimo (8 GB recomendado)

Almacenamiento

1 GB libre (SSD recomendado)

CPU

2 núcleos (4 con hyper-threading recomendados)

Versión de Windows

Windows 10/11 Pro, Enterprise o Education

Activación de Windows Sandbox

Método 1: A través del Panel de control

  1. Abre "Activar o desactivar características de Windows"
  2. Busca y selecciona "Windows Sandbox"
  3. Haz clic en Aceptar y reinicia el equipo cuando se te solicite

Método 2: Usando PowerShell

Ejecuta PowerShell como administrador y utiliza el siguiente comando:

Enable-WindowsOptionalFeature -FeatureName "Containers-DisposableClientVM" -All -Online

Uso de Windows Sandbox

Una vez activado, puedes iniciar Windows Sandbox desde el Menú Inicio. La aplicación se abrirá como una ventana independiente que muestra un escritorio aislado.

Características de uso
  • Puedes copiar archivos desde tu sistema principal y pegarlos en Sandbox
  • Navega por internet de forma segura para probar sitios web potencialmente riesgosos
  • Instala y prueba aplicaciones sin dejar rastro en tu sistema principal
  • Al cerrar Sandbox, todo el contenido se elimina automáticamente
Importante: Windows Sandbox no está disponible en la edición Home de Windows. Si tienes esta edición, necesitarás actualizar a una versión superior o usar alternativas como Sandboxie.

Casos de uso comunes

  • Prueba de software desconocido
  • Abrir archivos de fuentes no confiables
  • Navegación web segura
  • Pruebas de desarrollo de software
  • Visitar sitios web potencialmente maliciosos

Alternativas

  • Sandboxie-Plus
  • Máquinas virtuales con VirtualBox/VMware
  • Contenedores Docker
  • Hyper-V de Windows

Configuración avanzada

Puedes personalizar Windows Sandbox mediante archivos de configuración (.wsb) para:

  • Controlar el acceso a la red
  • Habilitar/deshabilitar el portapapeles compartido
  • Mapear carpetas específicas
  • Personalizar el entorno virtual

Solución de problemas

Si Sandbox no funciona:

  • Verifica que la virtualización esté habilitada en el BIOS/UEFI
  • Asegúrate de tener una edición compatible de Windows
  • Comprueba que tienes suficientes recursos del sistema
