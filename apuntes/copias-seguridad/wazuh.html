Wazuh

Wazuh es una plataforma unificada de seguridad de código abierto que combina múltiples funcionalidades de seguridad en una sola solución.

Características Principales

  • Detección de intrusiones (HIDS/NIDS)
  • Monitoreo de integridad de archivos (FIM)
  • Análisis de registros
  • Detección de vulnerabilidades
  • Cumplimiento normativo

Arquitectura

  • Wazuh Manager: Cerebro del sistema que procesa eventos
  • Wazuh Indexer: Almacena e indexa eventos (basado en OpenSearch)
  • Wazuh Dashboard: Interfaz web para visualización
  • Wazuh Agent: Recolector de datos en endpoints
Compatibilidad: Soporta Windows, Linux, macOS, AIX, Solaris y entornos cloud como AWS, Azure y GCP.
