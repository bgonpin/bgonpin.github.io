Pharming

Definición

El pharming es una técnica de phishing que redirige a los usuarios de sitios web legítimos a páginas falsas sin que la víctima sea consciente. A diferencia del phishing tradicional que requiere interacción del usuario, el pharming manipula la resolución de DNS o infecta dispositivos para interceptar el tráfico web.

Cómo Funciona

El pharming opera a nivel de infraestructura de red, interceptando las conexiones antes de que lleguen al usuario:

  • Manipulación DNS: Altera los servidores DNS para redirigir dominios legítimos a IPs maliciosas.
  • Malware local: Infecta dispositivos con software que modifica el archivo hosts o proxy settings.
  • Ataques a routers: Compromete routers domésticos o corporativos para redirigir todo el tráfico.
  • Envenenamiento de caché: Corrompe la caché DNS de proveedores de internet o resolvers locales.

Ejemplos Comunes

  • Un usuario intenta acceder a su banco, pero es redirigido a un sitio falso sin darse cuenta.
  • Visitas a sitios de comercio electrónico que muestran páginas fraudulentas para robar datos de tarjetas.
  • Acceso a portales gubernamentales que llevan a sitios de phishing para obtener información personal.
  • Redirecciones automáticas en redes WiFi públicas comprometidas.

Medidas de Prevención

  • Utilizar conexiones HTTPS y verificar certificados SSL.
  • Configurar DNS seguros y utilizar servicios como DNSSEC.
  • Mantener actualizado el software antivirus y anti-malware.
  • Ser cauteloso en redes públicas y utilizar VPN cuando sea necesario.
