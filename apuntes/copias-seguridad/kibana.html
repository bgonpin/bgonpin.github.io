Kibana

Kibana es una interfaz de visualización de datos de código abierto que permite explorar, visualizar y analizar grandes volúmenes de datos de manera interactiva.

Características Principales

  • Visualización de datos: Gráficos interactivos, dashboards y reportes en tiempo real
  • Búsqueda y filtrado: Búsqueda avanzada con filtros dinámicos
  • Dashboards personalizables: Creación de paneles de control adaptados a necesidades específicas
  • Análisis en tiempo real: Monitorización continua de datos y métricas
  • Integración con Elasticsearch: Funciona nativamente con Elasticsearch/OpenSearch

Componentes Clave

  • Discover: Exploración y búsqueda interactiva de datos
  • Visualize: Creación de gráficos y visualizaciones
  • Dashboard: Combinación de visualizaciones en paneles integrados
  • Canvas: Presentaciones dinámicas con elementos personalizados
  • Maps: Visualización de datos geoespaciales
Integración con Wazuh: Kibana se utiliza en Wazuh como interfaz principal para visualizar alertas de seguridad, logs, dashboards de cumplimiento y métricas del sistema.
