OpenSearch - Motor de Búsqueda y Analytics

OpenSearch es un motor de búsqueda y análisis de código abierto, bifurcado de Elasticsearch, que permite indexar, buscar y analizar grandes volúmenes de datos en tiempo real.

Características Principales

  • Búsqueda distribuida: Indexa y busca datos a través de múltiples nodos
  • Analytics en tiempo real: Proporciona análisis y agregaciones de datos
  • Escalabilidad horizontal: Crece agregando más nodos al clúster
  • RESTful API: Interfaz HTTP para interactuar con el motor
  • Plugins extensibles: Soporte para funcionalidades adicionales

Componentes Arquitectónicos

  • Índices: Estructuras de datos que almacenan documentos
  • Shards: Particiones de índices para distribución de datos
  • Réplicas: Copias de shards para alta disponibilidad
  • Nodos maestros: Gestionan el estado del clúster
  • Nodos de datos: Almacenan y procesan datos
Relación con Wazuh: OpenSearch es el componente central del Wazuh Indexer, responsable de almacenar y hacer consultables todos los eventos de seguridad generados por Wazuh.
