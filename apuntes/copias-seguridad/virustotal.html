VirusTotal

VirusTotal es una plataforma gratuita que analiza archivos y URLs sospechosos utilizando múltiples motores antivirus y herramientas de detección de malware.

Servicios Principales

  • Análisis de archivos: Escaneo de malware, troyanos, ransomware, etc.
  • Análisis de URLs: Detección de sitios web maliciosos y phishing
  • Análisis de dominios: Verificación de dominios sospechosos
  • API pública: Integración programática para automatización
  • Informes detallados: Resultados de múltiples motores antivirus

Motores Antivirus Integrados

  • Motores principales: Más de 70 compañías antivirus
  • Especializados: Herramientas específicas para diferentes tipos de malware
  • Comportamental: Análisis de comportamiento de archivos
  • De reputación: Verificación contra bases de datos de reputación
Integración con Wazuh: Wazuh puede integrarse con VirusTotal para enriquecer alertas de seguridad con información detallada sobre malware detectado.
