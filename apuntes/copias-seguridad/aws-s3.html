AWS S3 - Simple Storage Service

Amazon Simple Storage Service (Amazon S3) es un servicio de almacenamiento de objetos que ofrece escalabilidad, disponibilidad de datos, seguridad y rendimiento líderes del sector.

Características Principales

  • Almacenamiento de objetos: Almacena y recupera cualquier cantidad de datos desde cualquier ubicación
  • Escalabilidad ilimitada: Crece automáticamente para manejar grandes volúmenes de datos
  • Durabilidad del 99.999999999%: Diseñado para mantener datos seguros durante décadas
  • Clases de almacenamiento: Múltiples opciones para optimizar costos y rendimiento
  • Seguridad integrada: Cifrado, control de acceso y auditoría de acceso

Clases de Almacenamiento

  • S3 Standard: Para datos de acceso frecuente con alta durabilidad
  • S3 Intelligent-Tiering: Optimiza automáticamente costos moviendo datos
  • S3 Standard-IA: Para datos de acceso poco frecuente
  • S3 One Zone-IA: Para datos no críticos en una zona de disponibilidad
  • S3 Glacier: Para archivado de largo plazo con bajo costo

Características de Seguridad

  • Cifrado en reposo: SSE-S3, SSE-KMS, SSE-C para proteger datos
  • Control de acceso: Bucket policies, IAM policies, Access Points
  • Registro de acceso: Server Access Logging para auditoría
  • Bloqueo de objetos: WORM (Write Once Read Many) para cumplimiento
  • Versionado: Mantiene múltiples versiones de objetos
Importancia en Wazuh: Wazuh puede integrarse con AWS S3 para almacenar y analizar logs de acceso, backups de configuración, y datos de auditoría. S3 puede servir como repositorio central para logs de Wazuh, proporcionando durabilidad, escalabilidad y análisis de costos optimizado para grandes volúmenes de datos de seguridad.
