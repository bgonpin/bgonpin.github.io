Tutorial: Cómo Añadir una Ruta Estática Permanente en Windows

Este tutorial te guiará a través del proceso para agregar una ruta estática en tu sistema operativo Windows, garantizando que persista incluso después de reiniciar tu equipo.

Importante: Debes ejecutar el Símbolo del sistema o PowerShell con permisos de administrador para poder modificar la tabla de enrutamiento del sistema.

Paso 1: Abrir el Símbolo del sistema (CMD) como Administrador

Haz clic en el botón de inicio de Windows y busca "CMD" o "Símbolo del sistema". Haz clic derecho sobre el resultado y selecciona "Ejecutar como administrador".

Paso 2: Usar el comando route

El comando principal para añadir una ruta estática es route. Para que la ruta sea permanente, debes incluir el parámetro -p.

La sintaxis general del comando es la siguiente:

route -p add [destino] MASK [máscara_de_subred] [puerta_de_enlace]

Ejemplo Práctico

En este ejemplo, vamos a configurar una ruta para que el tráfico destinado a la red 10.0.0.0/24 sea dirigido a través de la puerta de enlace 192.168.1.12. La máscara de subred /24 se traduce en 255.255.255.0.

Introduce el siguiente comando en la ventana del Símbolo del sistema y presiona Enter:

route -p add 10.0.0.0 MASK 255.255.255.0 192.168.1.12
  • 10.0.0.0: La red de destino.
  • 255.255.255.0: La máscara de subred de la red de destino.
  • 192.168.1.12: La dirección IP de la puerta de enlace (el router o dispositivo que conoce el camino a la red de destino).

Paso 3: Verificar que la ruta se haya añadido

Puedes confirmar que la ruta se agregó correctamente a la tabla de enrutamiento persistente. Para ello, ejecuta el siguiente comando:

route print

Busca en la sección "Rutas persistentes" y deberías ver una entrada para la red 10.0.0.0.

Información Adicional

Para eliminar una ruta estática que ya no necesitas, utiliza el siguiente comando, reemplazando la red de destino y la puerta de enlace con los valores correctos:

route delete 10.0.0.0 MASK 255.255.255.0 192.168.1.12
